# CartPole-RL-vs-LQR
基于强化学习（PPO/REINFORCE/A2C）与经典控制理论（LQR）的倒立摆（CartPole-v1）控制对比实验。


```markdown
# 基于强化学习与经典控制理论的倒立摆控制对比实验
> **任务1实验报告（基于强化学习的倒立摆控制）开源代码仓库**

本项目在 Gymnasium `CartPole-v1` 经典控制环境下，系统性地对比了经典控制理论中的 **LQR（线性二次型调节器）** 与三种典型的深度强化学习算法——**PPO（近端策略优化）**、**REINFORCE（无基线蒙特卡洛策略梯度）** 以及 **A2C（优势演员-评论家）** 在不稳定非线性系统中的控制效果、收敛速度及渐进稳定性。

## 🌟 实验核心亮点

1. **长周期训练下的策略稳定性验证（No Early Stopping）**
   本实验全局取消了常规强化学习中“达到分数即停止”的早停机制，所有强化学习算法均被强制跑满 **1500 个回合（Episodes）**。该设定旨在极其严苛的长时间维度下，观测并验证单任务连续训练后期模型是否会出现性能退化或策略震荡问题。
2. **密集存活奖励机制优化（Dense Survival Reward）**
   对环境的终止奖励进行了自定义调整：全面废除部分场景下设定的额外失败惩罚（$-100$），保留纯粹的**每步 $+1$ 密集存活奖励**。此设计从底层平滑了价值估值网络（Critic）在逼近真实价值时的梯度分布，有效避免了由于终止状态负向惩罚突变引发的训练不稳定。
3. **LQR 物理建模极性修正与严谨对齐**
   在倒立摆系统的非线性动力学建模中，以竖直向上平衡点（$\theta \approx 0$）为工作点进行泰勒展开线性化。在提取状态方程 $\dot{x} = Ax + Bu$ 时，精确修正了输入矩阵 $B$ 中对应角加速度输入增益的物理惯性极性符号（即索引位置 `[3,0]` 元素），修正后的 $B$ 矩阵关键项表示为：
   $$B = \begin{bmatrix} 0 \\ \frac{1}{M_c} \\ 0 \\ -\frac{1}{M_c l} \end{bmatrix}$$
   该底层排坑操作赋予了 LQR 理论解析解层面的极致控制基准。

## ⚙️ 实验控制变量与统一设定

为保证算法对比的绝对公平性，本实验对所有强化学习算法进行了严格的控制变量对齐：
* **网络架构**：所有算法的 Actor 与 Critic 网络均统一采用两层全连接隐藏层（128 $\times$ 128），激活函数统一使用 `ReLU`。
* **优化器与学习率**：统一采用 `Adam` 优化器，基础学习率设定为 $1 \times 10^{-3}$，折扣因子统一为 $\gamma = 0.99$。
* **状态输入与环境**：统一采用 CartPole-v1 默认的连续状态输入，不作额外归一化处理；实验均固定相同的全局随机种子以确保可复现性。
* **各算法特性超参数**：
  * **PPO**：裁剪系数 `clip_param = 0.2`，微批次更新 `K_epochs = 8`，触发更新回合数 `update_episodes = 5`。
  * **REINFORCE**：设定为“无基线（No Baseline）”版本，作为高方差策略梯度的对照组。
  * **A2C**：限定在纯 1-step TD 且无 $n$ 步回报设定的极简架构下运行。

## 🚀 复现与运行指南

### 1. 环境依赖配置

推荐使用 Python 3.9+ 环境。在同级目录下执行以下命令安装核心依赖：

```bash
pip install -r requirements.txt
# 或手动安装: pip install gymnasium==0.29.1 numpy==1.26.4 torch==2.3.0 matplotlib==3.8.4

```

### 2. 执行训练与测试

直接运行对应的 Python 脚本即可开始 1500 个回合的训练与验证，并在终端实时打印进度：

```bash
python ppo.py

```

*(算法运行完毕后，会自动在当前目录下生成/覆盖对应的 `_metrics.csv` 数据文件)*

### 3. 数据可视化

将生成的 CSV 数据文件与 MATLAB 绘图脚本保持在同一目录下，运行 `reward_paint.m` 或 `entropy_paint.m`，即可一键复现报告中的综合对比图表。

## 📊 实验结果简述

*(注：以下强化学习算法表现，均为关闭探索、直接取最大概率动作的**确定性策略测试**结果)*

1. **LQR 控制基准**：无需网络训练，基于 $\theta \approx 0$ 工作点的解析解在 1500 回合测试中局局精准保持 500 分满分，提供绝对的横向对比基线。
2. **PPO 近端策略优化**：展现出极高的样本效率。在 200~300 回合内即快速收敛，策略熵平滑衰减至 0.2~0.3 区间，对应策略从随机探索成功转向稳定的确定性决策，最终测试性能完美重合于 LQR 基准。
3. **REINFORCE（无基线版）**：受蒙特卡洛高方差特性的理论限制，收敛速度较慢（约 800~1000 回合逼近上限），收敛后曲线存在一定波动，但在长周期下最终能维持有效控制。
4. **A2C（单步时序差分版）**：在本实验纯 1-step TD 且无多步自举（$n$-step return）的严苛设定下，优势函数的估计存在较大偏差，导致模型在长周期训练中出现严重的不稳定与震荡，未能收敛至理想控制状态。
