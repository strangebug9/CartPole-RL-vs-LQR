# CartPole-RL-vs-LQR
基于强化学习（PPO/REINFORCE/A2C）与经典控制理论（LQR）的倒立摆（CartPole-v1）控制对比实验。

```markdown
# 基于强化学习与经典控制理论的倒立摆控制对比实验
> **任务1实验报告（基于强化学习的倒立摆控制）开源代码仓库**

本项目在 Gymnasium `CartPole-v1` 经典控制环境下，系统性地对比了经典控制理论中的 **LQR（线性二次型调节器）** 与三种典型的深度强化学习算法——**PPO（近端策略优化）**、**REINFORCE（蒙特卡洛策略梯度）** 以及 **A2C（优势演员-评论家）** 在不稳定非线性系统中的控制效果、收敛速度及渐进稳定性。

## 🌟 实验核心亮点

1. **严格的渐进稳定性评估（No Early Stopping）**
   全局取消了常规强化学习中“达到分数即停止”的早停机制，所有强化学习算法均被强制跑满 **1500 个回合（Episodes）**。该设定旨在极其严苛的长时间维度下，测试算法在后期的抗灾难性遗忘（Catastrophic Forgetting）能力。
2. **稀疏奖励工程（Sparse Reward）**
   全面废除原环境设定中的 `-100` 失败惩罚，重构为纯粹的**每步 `+1` 稀疏奖励**。此设计从底层保护了价值估值网络（Critic）在逼近真实价值时的平滑性，有效避免了负向断崖惩罚引发的梯度爆炸。
3. **LQR 物理建模极性修正**
   在倒立摆系统的非线性动力学泰勒展开建模中，精确修正了状态输入矩阵 $B$ 中第 4 行 $[3,0]$ 元素的物理惯性极性符号错误（精准修正为 $-\frac{1}{M_c l}$），赋予了 LQR 解析解层面的极致控制基准。

## 🚀 运行指南

### 1. 环境依赖

确保你的 Python 环境中已安装以下核心依赖：

```bash
pip install gymnasium numpy torch

```

### 2. 执行训练与测试

直接运行对应的 Python 脚本即可开始 1500 个回合的训练/测试，并在终端实时打印进度：

```bash
python ppo.py

```

*(算法运行完毕后，会自动在同级目录下生成/覆盖对应的 `_metrics.csv` 数据文件)*

### 3. 数据可视化

将生成的 `.csv` 数据文件与 `.m` 脚本置于同一目录下，使用 MATLAB 运行 `reward_paint.m` 或 `entropy_paint.m`，即可一键复现报告中的高清对比图表。

## 📊 实验结果简述

* **LQR 控制基准**：无需任何网络训练，通过代数黎卡提方程求得解析解。在 1500 回合测试中局局精准保持 500 分满分，展现出绝对的稳定性。
* **PPO 近端策略优化**：展现出恐怖的样本效率。在 200~300 回合内即收敛并稳定锁死 500 分，策略熵平滑衰减至安全置信度（0.2~0.3），曲线后续完美重合于 LQR 基准。
* **REINFORCE 经典策略梯度**：受蒙特卡洛高方差特性的物理限制，收敛较慢（约 800~1000 回合摸到上限），收敛后曲线伴随轻微震荡，但整体平稳。
* **A2C 单步时序差分**：在纯 `1-step TD` 与稀疏奖励双重作用下，早期自举偏差持续累积，导致模型在后续训练中发生灾难性遗忘，无法收敛。
